Die Hard Trilogy


Works
> (USA 1.1 - Greatest Hits) http://redump.org/disc/2011/
> (Europe) http://redump.org/disc/894/


Converts justifiers to dual guncon



Instructions:
1. Rip game cd to bin/cue or ccd/img format
2. Run bin/ppf-o-matic3
3. Drag-drop bin or img file
4. Drag-drop bin/ppf file
5. Hit patch



Uninstall:
1. Run bin/ppf-o-matic3
2. Click 'undo patch'
3. Drag-drop bin or img file
4. Drag-drop bin/ppf file
5. Hit patch



NOTE:
- Hold Trigger + Aux = fire rockets
